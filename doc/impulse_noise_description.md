# Принцип создания импульсного шума

Импульсный шум - это высококонтрастный шум, представляющий собой изолированные элементы изображения с максимальным уровнем яркости (255) или нулевым (0).

## Типы импульсного шума

### 1. "Соль" (Salt)
- Пиксели шума имеют максимальную яркость (255)
- Представляют собой белые точки на изображении

### 2. "Перец" (Pepper)
- Пиксели шума имеют нулевую яркость (0)
- Представляют собой черные точки на изображении

### 3. "Соль и перец" (Salt & Pepper)
- Комбинация обоих типов шума
- Некоторые пиксели имеют яркость 255, другие - 0

## Интенсивность шума

### Точечный
- Шум представлен отдельными пикселями
- Распределены случайным образом по изображению

### Строковый
- Шум представлен линиями из пикселей
- Длина строковой помехи от 2 до 5 пикселей

## Математическое описание

### Для "Соли" (точечный)
Количество пикселей шума рассчитывается по формуле:
```
KOL = η * B_ИСХ / 255
```

### Для "Перца" (точечный)
Количество пикселей шума рассчитывается по формуле:
```
KOL = η * B_ИСХ
```

### Для "Соль и перец"
Количество пикселей шума рассчитывается по формуле:
```
KOL = η * B_ИСХ / MO_Ш
```

Где:
- `η` - уровень шума (0.25, 0.5, 0.75)
- `B_ИСХ` - энергия исходного изображения
- `MO_Ш` - математическое ожидание шумового поля

## Алгоритм

1. **Выбор типа шума**: Определяется вид импульсного шума ("соль", "перец" или "соль и перец") и его интенсивность ("точечный" или "строковый").

2. **Расчет количества пикселей шума**: По соответствующей формуле вычисляется количество пикселей, которые должны быть изменены.

3. **Генерация координат**: С помощью генератора случайных чисел формируются координаты пикселей шума.

4. **Применение шума**: 
   - Для точечного шума - устанавливаются значения яркости для отдельных пикселей
   - Для строкового шума - формируются линии из пикселей с одинаковой яркостью

## Особенности реализации

- Используется генератор случайных чисел для определения координат шума
- Поддерживается три уровня интенсивности шума: 0.25, 0.5, 0.75
- Реализованы оба типа интенсивности: точечный и строковый
- Для "соль и перец" тип шума для каждого пикселя определяется сравнением с математическим ожиданием шумового поля
