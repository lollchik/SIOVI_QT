# Принцип создания аддитивного шума

Аддитивный шум - это тип шума, который добавляется к исходному изображению. В результате получается изображение, искаженное шумом.

## Математическое описание

Яркость элемента изображения, искаженного аддитивным шумом с заданным уровнем энергии, рассчитывается следующим образом:

```
pix = f_ИСХ(x,y) + η * (f_Ш(x,y) - MO_ш) + ν
```

Где:
- `f_ИСХ(x,y)` - яркость пикселя исходного изображения
- `η` - уровень шума (0.25, 0.5, 0.75)
- `f_Ш(x,y)` - шумовое поле
- `MO_ш` - математическое ожидание шумового поля
- `ν` - накопленная ошибка (error diffusion)

## Алгоритм

1. **Генерация шумового поля**: Создается шумовое поле размером 512x512 со случайными значениями от 0 до 255.

2. **Вычисление математического ожидания**: Рассчитывается среднее значение шумового поля:
   ```
   MO_ш = (1/N²) * ΣΣ f_Ш(x,y)
   ```

3. **Расчет энергии исходного изображения**: 
   ```
   B_исх = ΣΣ [f_исх(x,y)]²
   ```

4. **Расчет целевой энергии шума**:
   ```
   B_ш = η * B_исх
   ```

5. **Применение шума**: Для каждого пикселя применяется формула с контролем диапазона [0, 255] и переносом ошибки (error diffusion).

## Особенности реализации

- Используется равномерное распределение случайных чисел
- Реализован механизм переноса ошибки для сохранения энергии шума
- Поддерживается воспроизводимость результатов через seed генератора случайных чисел
