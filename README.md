# SIOVI_QT
Siovi pictures optimization Visualizator
## Первоначальные действия при получении программы и необходимости пересборки
### Необходимые пакеты для установки
    sudo apt install build-essential qtbase5-dev qt5-qmake libqt5widgets5

### Сборка и запуск проекта
    cd build 
    cmake ..
    make
    ./INSTALLER

# !!! ВАЖНО: для корректной работы скрипта должны быть настроены зеркала в системе т.е. корректно прописанный файл (/etc/apt/sources.list)
